What it does

Guardian App is a computer vision project that allows for paralyzed patients to use the motions of the eyes to send messages to their nurse or caregiver.

How we built it

Guardian was built using Python, Flask, OpenCV, and HTML/CSS.

Challenges we ran into

Attempting to construct the FaceRecogniton as there was alot of factors that needed to be taken into account

Accomplishments that we're proud of

Getting the FaceRecogniton to work and being able to change the text of the boxes

What we learned

We learned a considerable amount of OpenCV from python, which will help us with forming better FaceRecogniton

What's next for Guardian

Add a notifcation feature when the patient is selecting an object and allow a person to look/blink to select a box
